# return / devolutivas (logs de handoff)

Pasta para **ping‑pong** entre ambientes (ex.: Cursor ↔ Claude Code): decisões, outputs validados e checklists sem depender do histórico do chat.

- Cada `log-*.md` é um recorte datado ou por tema.
- Evolution `fetchInstances` 200: `log-2026-04-09-evolution-fetchinstances-200.md`.
- Fix QR só `count`: `log-2026-04-09-evolution-qr-count0-fix.md`.
- DB Evolution Railway: `log-2026-04-09-evolution-railway-db-env.md`.
- Ao fechar um sprint, adicionar um novo arquivo ou anexar seção no handoff consolidado.

**Papel sugerido:** codificação/revisão no Cursor; governança, síntese e continuidade no Claude Code — ambos alimentam esta pasta.
